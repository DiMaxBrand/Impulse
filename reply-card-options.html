Implementation note

A, B, C, D are all implementable today in the existing View system with no Compose dependency. Option E's spring morph is demoed here in CSS/JS but would use Modifier.bounceClick() or AnimatedShape in Jetpack Compose, and can be back-ported to Views via a custom ViewPropertyAnimator with an overshoot interpolator. The current code uses Option A.
